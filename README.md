# lagou
AngularJS仿拉勾网WebApp  开发移动端单页应用
## 技术栈：
- angular(1.5.8)+gulp+bower+less+ui.router
## 预览效果(无法在线预览，可以clone到本地运行)
- 先后运行npm install,bower install。之后执行gulp命令即可在浏览器查看效果，只支持移动端。（后台数据用静态json数据模拟）
